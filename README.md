# OCR: Boletim de Catanduva

Implementação de solução de reconhecimento de caracteres para melhor acompanhamento epidemiológico na cidade de Catanduva.

### Requisitos:
- Python 2.7
- - pytesseract
- - numpy
- - BeautifulSoup
- OpenCV 4
- Tesseract 4.0

OU

- C++
- OpenCV 4
- Tesseract 4.0


### Funcionamento:
- Site da prefeitura é acessado
- Boletim do dia é baixado
- Imagem é pré-processada
- Dados são extraídos e salvos
